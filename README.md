# Smart Bharat – AI Civic Companion

Smart Bharat is an AI-powered civic companion prototype designed to simplify access to government services, welfare schemes, civic issue reporting, complaint tracking, and civic guidance.

## Features

- AI Civic Assistant
- Government Scheme Finder
- Personalized Scheme Recommendations
- Public Issue and Grievance Reporting
- AI-based Complaint Analysis
- Complaint Tracking
- Citizen Dashboard
- Multilingual Interface
- Emergency and Citizen Support Information

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Bolt.new

## Live Demo

Smart Bharat is deployed as a working web prototype.

## Disclaimer

Smart Bharat is a prototype and is not an official Government of India service. Users should verify important information and scheme eligibility through official government portals.
